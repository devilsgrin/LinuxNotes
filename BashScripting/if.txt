Bu yazıda bash dilindeki if yapısından ve if komutunun 
tüm parametrelerinden bahsedeceğim.

Örnek kullanım:

#!/bin/bash

DAY="Monday"

if [ $DAY = "Monday" ]; then
    echo "Today:" $DAY
elif [ $DAY == "Tuesday" ]; then
    echo "Today:" "Tuesday"
else
    echo "Day:" "I don't know really, what today from days"
fi

Burada dikkat edilmesi noktalardan birisi; [] içindeki koşulun,
kapalı parantezlere yapışık olmaması. Koşulun başlangıcı ya da
sonu, kapalı paranteze değerse, syntax error alınır.
 
Bunun yanında, her durumda [] parantez kullanmak zorunda değilsiniz.
Dümdüz bir kullanım da yapabilirsiniz.

"=" ile "==" arasındaki dişe dokunur bir fark yok ama, "==" kullanılması,
posix açısından daha uyumludur ama aman allah'da bir fark yok yani.

Aşağıdaki örnekte, iç içe if koşullarının kullanımını gösterdim:

#!/bin/bash

NUMBER=2
REAL_NUMBER=3

if [ $NUMBER -eq 2 ]; then
    if [ $NUMBER -lt 5 ]; then
	echo "Your Real Number:" $REAL_NUMBER
    fi
elif [ $NUMBER == 8 ]; then
    echo "Your Real Number:" $REAL_NUMBER
else
    echo "No number"
fi

Burada "-eq" "-lt" gibi özel operatörlere takılmayın, bunların
"cheeat sheat"ini yazının en sonunda vereceğim. 

Dikkat edilmesi gereken nokta; ilk "if"in içindeki, "if"den sonra
"fi" komutunun gelmesidir. Yoksa syntax error verecektir. 
İlk "if"i ve sonrasındaki koşulları kapsayan ( ikinci "if" harici )
"fi" komutu, en sondadır. Bunlara dikkat ederseniz rahat edersiniz ;)

Örnekler çoğaltılabilir, şimdilik gerek görmüyorum. Şimdi aşağıya,
"if" içinde kullanabileceğiniz bütün operatörleri şutluyorum.
